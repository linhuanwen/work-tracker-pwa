# 工作清单 PWA

> 国企人力资源经办人的个人工作管理系统 —— 每日任务管理、周/月工作小结自动生成、年度述职报告按维度归纳。

## 为什么做这个

年底述职、评优、晋升时，往往需要回顾一整年的工作成果。但日常忙起来很少记录，到年底只能凭记忆拼凑，容易遗漏关键业绩。

这个工具把"记录"融入日常流程——每天管理 to-do 的时候顺手记录量化产出，周末一键生成周小结，月底自动聚合月报，年底六个维度自动归纳年度报告。**数据只录一次，报告随时可出。**

## 核心功能

### 📋 每日任务管理
- 优先级三级（🔴紧急 / 🟡重要 / 🟢日常），紧急任务独立顶置区最多 5 条
- 长期项目子任务拆解 + 自动进度条（3/8 完成），推进感可量化
- 领导交办特殊标记（来源、交办时间、时限），不遗漏上级交代事项
- 多项量化产出录入（如"审查 15 人次""助力晋升 3 人""修订制度 2 份"）
- 跨年任务休眠：平时不占空间，临近截止前 1-2 个月自动激活

### 📊 周小结（每周自动生成）
四段式模板：本周完成任务 → 长期项目推进 → 下周计划 → 需协调事项

- 下周计划手动补充的条目自动回填为任务清单新任务
- 一键复制纯文本，粘贴到 OA / 邮件

### 📈 月度小结
- 当月量化产出汇总表（按分类聚合，如"本月审查 45 人次，助力晋升 3 人"）
- 项目进度月度回顾 + 月度反思（手写）+ 下月重点

### 📝 年度报告（六大维度归纳）
按国企人力工作板块自动归纳：
1. **人员配置** — 调配方案、编制调整等
2. **内部招聘 / 晋升晋等** — 资格审查、答辩辅导、助力晋升
3. **奖惩管理** — 奖惩执行情况
4. **绩效管理** — 考核组织、数据汇总、结果反馈
5. **劳动关系** — 合同续签、纠纷处理等
6. **领导交办** — 上级交办事项完成情况

每个维度自动汇总：量化产出合计 + 里程碑节点 + 附表（月度趋势 + 全年量化总表）。每个维度有手写"关键业绩提炼"区域。

### 🤖 AI 润色（可选）
- 个人电脑上运行独立 Python 脚本，调用 AI API 润色小结语言
- 预设"国企人力资源文风"——正式、简洁、结构化、用数据说话
- AI 润色结果始终可手动编辑，你保留最终修改权

## 技术方案

### 为什么选 PWA

国企电脑通常有限制：不能装 exe、有白名单管控、远程监控。PWA（Progressive Web App）就是浏览器打开的网页，可以"安装"成独立桌面窗口——不需要任何安装权限，也不会被监控软件标记。

| 约束 | 方案 |
|------|------|
| 不能装软件 | PWA 在浏览器中运行，安装为独立窗口无需管理员权限 |
| 远程监控 | 浏览器和 WPS Office 都是白名单办公软件 |
| 电脑手机同步 | WPS 云文档自动同步 data.json |
| 电脑只能上白名单网站 | AI 润色在个人电脑上跑，不依赖公司网络 |

### 架构

```
┌─────────────────────┐          ┌──────────┐          ┌──────────────────┐
│  公司电脑             │          │ WPS 云端  │          │  个人电脑          │
│                     │          │          │          │                  │
│  PWA（浏览器独立窗口） ── 读写 ──→ data.json ←── 同步 ──→ Python AI 润色脚本 │
│                     │          │          │          │                  │
└─────────────────────┘          └────┬─────┘          └──────────────────┘
                                     │
                                     │ 同步
                                ┌────▼─────┐
                                │ 安卓手机   │
                                │ PWA      │
                                └──────────┘
```

- **前端**：React 18 + Vite + TypeScript + vite-plugin-pwa
- **数据**：单一 `data.json`，浏览器 File System Access API 读写，WPS 云文档文件夹自动跨设备同步
- **AI 脚本**：独立 Python 脚本，命令行运行，支持 DeepSeek / 通义千问
- **零后端**：不需要服务器、不需要数据库、不需要部署

### 数据流

1. 你在公司电脑或手机上用 PWA 管理任务
2. PWA 读写 WPS 云文档文件夹里的 `data.json`
3. WPS 自动同步到云端，再同步到其他设备
4. 周末/月末，在个人电脑上运行 AI 润色脚本
5. 润色结果写回 `data.json`，自动同步回公司电脑

## 使用前准备

1. **公司电脑**：安装 WPS Office，登录 WPS 账号（开通云文档）
2. **个人电脑**：同样安装 WPS Office，登录同一账号
3. **安卓手机**：安装 WPS App，登录同一账号
4. 打开 PWA → 浏览器菜单 → "安装" → 独立窗口钉到任务栏
5. （可选）个人电脑上配置 Python + AI API key，用于润色功能

## 分类体系

预设七类（可在设置页随时调整）：

| 分类 | 典型工作内容 |
|------|-------------|
| 人员调配 | 编制调整、人员配置方案、岗位分配 |
| 内部招聘 / 晋升晋等 | 资格审查、材料辅导、答辩组织、助力晋升 |
| 奖惩管理 | 奖励申报、处分执行、结果归档 |
| 绩效管理 | 考核组织、数据汇总、结果反馈、面谈记录 |
| 劳动关系 | 合同续签、入离职办理、纠纷处理 |
| 领导交办 | 上级交办事项（有特殊标记和时限） |
| 其他 | 不归属以上分类的日常工作 |
