# 工作清单 — 个人任务管理与工作日志 PWA

> 每日任务管理 + 周/月工作小结自动生成 + 年度报告按维度归纳。**数据只录一次，报告随时可出。**

## 为什么做这个

日常忙起来很少记录，到年底回顾时只能凭记忆拼凑，容易遗漏重要工作成果。

这个工具把"记录"融入日常流程——每天管理 to-do 的时候顺手记录产出，周末一键生成周小结，月底自动聚合月报，年底自动归纳年度报告。

## 核心功能

### 📋 每日任务管理
- 优先级三级（🔴紧急 / 🟡重要 / 🟢日常），紧急任务独立顶置区最多 5 条
- 长期项目子任务拆解 + 自动进度条，推进感可量化
- 特殊标记（来源、交办时间、时限），不遗漏重要事项
- 多项量化产出录入（如"完成 3 项""处理 15 件"等）
- 跨年任务休眠：平时不占空间，临近截止前自动激活

### 📊 周小结（每周自动生成）
四段式模板：本周完成任务 → 长期项目推进 → 下周计划 → 需协调事项

- 下周计划手动补充的条目自动回填为任务清单新任务
- 一键复制纯文本，粘贴到邮件或 OA

### 📈 月度小结
- 当月量化产出汇总表（按分类聚合）
- 项目进度月度回顾 + 月度反思（手写）+ 下月重点

### 📝 年度报告
按工作板块自动归纳，每个板块汇总：
- 量化产出合计 + 里程碑节点
- 附表（月度趋势 + 全年量化总表）
- 手写"关键业绩提炼"区域

### 🤖 AI 润色（可选）
- 调用 AI API 润色小结语言，使表达更正式、简洁、结构化
- 润色结果始终可手动编辑，保留最终修改权
- 支持 DeepSeek / 通义千问等多种 API

## 技术方案

### PWA（Progressive Web App）

PWA 即浏览器打开的网页，可"安装"成独立桌面窗口——不需要安装权限。

| 特性 | 说明 |
|------|------|
| 零安装 | 浏览器中运行，安装为独立窗口无需管理员权限 |
| 离线可用 | Service Worker 缓存，断网也能正常使用 |
| 跨设备同步 | 数据文件放在云同步文件夹中自动跨设备同步 |
| 桌面体验 | 独立窗口、系统托盘、无边框透明窗口 |

### 架构

```
┌──────────────────┐       ┌──────────┐       ┌──────────────────┐
│   Windows 桌面    │       │ 云盘同步  │       │  其他设备          │
│                  │       │          │       │                  │
│  桌面启动器 (exe)  │── 读写 ──→ data.json ←── 同步 ──→ 手机/其他电脑    │
│  PWA 独立窗口     │       │          │       │                  │
└──────────────────┘       └──────────┘       └──────────────────┘
```

- **前端**：React 18 + Vite + TypeScript + vite-plugin-pwa
- **数据**：单一 `data.json`，云同步文件夹自动跨设备同步
- **桌面启动器**：Python + pywebview（WebView2），无边框透明窗口 + 系统托盘
- **AI 脚本**：独立 Python 脚本，命令行运行，支持 DeepSeek / 通义千问
- **零后端**：不需要服务器、不需要数据库、不需要部署

## 快速开始

### 方式一：直接使用预打包 exe

1. 下载 `release/` 文件夹到本地
2. 确保系统已安装 [WebView2 Runtime](https://developer.microsoft.com/en-us/microsoft-edge/webview2/)（Windows 11 已内置）
3. 双击 `工作清单.exe` 运行
4. 首次运行选择数据文件存放位置（建议放在云同步文件夹中）

### 方式二：PWA 模式（浏览器）

```bash
# 安装依赖 & 构建
npm install
npm run build

# 启动本地服务器
python serve.py
# 或: npx serve dist
```

然后浏览器打开 `http://127.0.0.1:5173`，浏览器菜单 → "安装" → 钉到任务栏。

### 方式三：从源码运行桌面版

```bash
# 前端构建
npm install
npm run build

# Python 依赖
pip install pywebview pystray Pillow python-docx requests

# 启动
python launcher.py
```

详细说明见 [docs/INSTALL.md](docs/INSTALL.md)。

## 数据同步

将 `data.json` 放在云同步文件夹（如 WPS 云文档、OneDrive、坚果云等）中，即可自动跨设备同步。

1. 在云同步文件夹中新建 `工作清单` 子文件夹
2. 在应用设置中选择该文件夹
3. 其他设备（手机/电脑）的 PWA 指向同一文件，数据自动同步
